# HomeFit Pro Rebuild

Workout-only Push/Pull/Legs tracker for Expo.

## What changed
- Proper Start Workout mode
- Complete / Log Workout button
- Add Set and Remove Set per exercise
- Mark sets and exercises done
- Smart Swap: shows closest replacements first
- Equipment creator with max kg and increment fields
- Half-kg dumbbell progression support
- Workout notes and exercise notes
- Save/load workout templates
- Exercise statistics page
- Fixed default priorities: shoulders, biceps, triceps

## GitHub / EAS build
Upload all files to your GitHub repo root, then run your EAS GitHub Action or use:

```bash
npx eas build -p android --profile preview
```

Preview profile builds an APK.
